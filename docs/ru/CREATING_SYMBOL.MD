# Создание символа с использованием меню примитивов

В этом руководстве описывается процесс создания нового изометрического символа (SKEY) с использованием геометрических примитивов в редакторе OpenIso. В качестве примера мы создадим символ задвижки (Gate Valve), который обычно состоит из двух треугольников, соединенных вершинами («бабочка»).

## Расположение инструментов

Инструменты рисования находятся на панели инструментов слева от основной области рисования (холста).
Основные кнопки, которые нам понадобятся:

*   ![Простые формы](images/menu_primitive.png) **Рисовать примитив (Draw Primitive)**: Кнопка с выпадающим меню для выбора геометрических фигур (треугольник, круг, прямоугольник и т.д.).
*   ![Точки соединения](images/point_arrive.png) **Точка соединения (Draw Connection Point)**: Для добавления точек входа и выхода трубопровода.
*   ![Шпиндель](images/point_spindle.png) **Точка шпинделя (Draw Spindle Connection Point)**: Для указания расположения привода или штурвала.
*   ![Сохранить](images/save.png) **Сохранить (Save)**: В верхней панели инструментов для сохранения созданного символа.

## Пошаговое создание символа задвижки (Gate Valve)

### Шаг 1: Подготовка нового SKEY

1.  Если у вас открыт существующий символ, нажмите правой кнопкой мыши в дереве SKEY и выберите **Создать новый SKEY** (или очистите текущее поле ввода).
2.  В панели свойств справа введите имя нового SKEY в поле **Код SKEY** (например, `VVFL`).
3.  Укажите описание (например, `Задвижка фланцевая`) и выберите тип фитинга (например, `Valve`).

### Шаг 2: Рисование тела символа (Треугольники)

Задвижка схематично изображается как два треугольника, соприкасающихся вершинами в центре.

1.  На панели инструментов рисования нажмите на кнопку **Рисовать примитив** (или стрелку рядом с ней).
2.  В появившемся меню раскройте группу **Базовые фигуры (Basic Shapes)**.
3.  Выберите инструмент **Треугольник (Triangle)**.
4.  Нарисуйте левую часть задвижки:
    *   Кликните левой кнопкой мыши на холсте, чтобы поставить первую точку (например, левый верхний угол).
    *   Кликните второй раз, чтобы поставить вторую точку (например, левый нижний угол).
    *   Кликните третий раз в центре символа, чтобы замкнуть треугольник (вершина).
    *   *Совет: Используйте сетку и привязку для точности.*

5.  Повторите действие для правой части:
    *   Инструмент **Треугольник** остается активным.
    *   Кликните в центре символа (в той же точке, где закончился первый треугольник).
    *   Кликните, чтобы поставить верхний правый угол.
    *   Кликните, чтобы поставить нижний правый угол.

Теперь у вас есть геометрическая основа символа в виде «бабочки».

### Шаг 3: Добавление точек соединения

Символ должен знать, где к нему присоединяется труба. Для этого нужны точки соединения (Connection Points).

1.  Нажмите кнопку **Рисовать точку соединения (Draw Connection Point)** на панели инструментов.
    *   По умолчанию выбирается **Точка входа (Arrive Point)** (зеленая).
2.  Кликните в середину левого основания первого треугольника, чтобы установить точку входа.
3.  Чтобы поставить точку выхода, нужно сменить тип точки (если это не происходит автоматически):
    *   В контекстном меню (правая кнопка мыши) или повторным выбором инструмента выберите **Точка выхода (Leave Point)** (красная).
4.  Кликните в середину правого основания второго треугольника.

### Шаг 4: Добавление точки шпинделя (Опционально)

Для клапанов и задвижек важно указать, где находится штурвал или привод, чтобы программа могла правильно ориентировать его при генерации изометрии.

1.  Нажмите кнопку **Рисовать точку шпинделя (Draw Spindle Connection Point)**.
2.  Установите точку в центре символа (на стыке двух треугольников).
3.  При необходимости вы можете выбрать ориентацию шпинделя по умолчанию.

### Шаг 5: Сохранение

1.  Убедитесь, что символ выглядит корректно.
2.  Нажмите кнопку **Сохранить (Save)** на верхней панели инструментов.
3.  Новый SKEY появится в дереве символов и будет сохранен в базе данных.

## Советы

*   **Отмена действий**: Если вы ошиблись при рисовании, используйте кнопку **Отменить (Undo)** или сочетание клавиш `Ctrl+Z`.
*   **Редактирование**: Вы можете выделить нарисованный примитив инструментом **Выбрать элемент (Select Element)** и перетащить его узловые точки для изменения формы.
*   **Точность**: Следите за координатами в строке состояния внизу окна для точного позиционирования точек.
